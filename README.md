# BMW Sales Analysis

data-science BMW sales experiments: data cleaning, exploration, statistical analysis, and visualization.

## Usage
- Clean data: `python src/data_cleaning.py` (writes `data/BMW_sales_data_cleaned.csv`).
- Explore data: `python src/data_exploration.py` (initial data overview and distributions).
- Statistical analysis: `python src/statistical_analysis.py` (correlations, hypothesis testing, trends).
- Visualizations: `python src/visualization_analysis.py` (charts, dashboards, interactive plots).

## What's in this folder
- `data/` — source datasets (`BMW_sales_data.csv`, `BMW_sales_data_cleaned.csv`).
- `src/` — scripts: `data_cleaning.py`, `data_exploration.py`, `statistical_analysis.py`, `visualization_analysis.py`.
- `notebooks/` — analysis notebooks (`analysis.ipynb`, `data_cleaning.ipynb`, `data_exploration.ipynb`, `visualization.ipynb`).
- `sql/` — SQL queries for business intelligence and exploratory analysis.
